# Smart Filing Database System (SQL)

## Overview
This project will create and deploy a relational database system to aid client strategy processes at Targeted Victory, a data-driven digital marketing and advocacy agency. The system will help the analysts to better manage client projects, structure research and develop insights in a high pace world where there is a strict turnaround.

The key data in the database are clients, projects, issues, industries, team members, and data sources, which are brought together in a structured schema. The system helps minimize manual data collection, eliminates redundancy, and facilitates quicker and more precise decision-making by organizing this information in one place.

## What We Built
- Designed a relational database schema (ERD)
- Created tables with primary and foreign keys
- Inserted sample data
- Wrote SQL queries using JOIN, GROUP BY, and filtering
